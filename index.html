GOOGLE SHEETS DATABASE
Connect a Google Sheet to persist data across devices. See setup guide below.
HOW TO SET UP YOUR DATABASE
1. Go to Google Sheets → create a new spreadsheet
2. Go to Extensions → Apps Script
3. Paste the Apps Script code (tap below to copy)
4. Click Deploy → New Deployment → Web App
5. Set access to "Anyone" → Deploy
6. Copy the URL and paste above
INSTALL AS APP
iPhone: Tap Share → Add to Home Screen
Android: Tap ⋮ → Install App / Add to Home Screen
Desktop: Click install icon in URL bar
